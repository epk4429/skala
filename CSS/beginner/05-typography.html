05. 타이포그래피 (Typography)

1. font-family (글꼴)

텍스트에 사용할 글꼴을 지정합니다.

/* 단일 글꼴 */ .text { font-family: Arial; } /* 여러 글꼴 (폴백) */ .text { font-family: "Malgun Gothic", "Apple SD Gothic Neo", sans-serif; } /* 글꼴 종류 */ .serif { font-family: serif; } /* 명조체 */ .sans-serif { font-family: sans-serif; } /* 고딕체 */ .monospace { font-family: monospace; } /* 고정폭 */ .cursive { font-family: cursive; } /* 필기체 */ .fantasy { font-family: fantasy; } /* 장식체 */

serif (명조체) - Serif 글꼴입니다.

sans-serif (고딕체) - Sans-serif 글꼴입니다.

monospace (고정폭) - Monospace 글꼴입니다.

Malgun Gothic - 맑은 고딕 글꼴입니다.

2. font-size (글자 크기)

텍스트의 크기를 설정합니다.

/* 픽셀 단위 */ .text { font-size: 16px; } /* em 단위 (부모 요소 기준) */ .text { font-size: 1.5em; /* 부모의 1.5배 */ } /* rem 단위 (루트 요소 기준) */ .text { font-size: 1.5rem; /* 루트의 1.5배 */ } /* 퍼센트 */ .text { font-size: 150%; } /* 키워드 */ .small { font-size: small; } .medium { font-size: medium; } .large { font-size: large; }

12px - 작은 텍스트

16px - 기본 텍스트

24px - 큰 텍스트

1.5em - 부모의 1.5배

1.5rem - 루트의 1.5배

3. font-weight (글자 굵기)

텍스트의 굵기를 설정합니다.

/* 키워드 */ .normal { font-weight: normal; } /* 400 */ .bold { font-weight: bold; } /* 700 */ /* 숫자 값 */ .thin { font-weight: 100; } /* 얇음 */ .light { font-weight: 300; } /* 가벼움 */ .normal { font-weight: 400; } /* 보통 */ .medium { font-weight: 500; } /* 중간 */ .bold { font-weight: 700; } /* 굵음 */ .bolder { font-weight: 900; } /* 매우 굵음 */

100 - 얇은 텍스트

300 - 가벼운 텍스트

400 - 보통 텍스트

700 - 굵은 텍스트

900 - 매우 굵은 텍스트

4. font-style (글자 스타일)

텍스트의 스타일을 설정합니다.

/* 일반 */ .normal { font-style: normal; } /* 이탤릭 */ .italic { font-style: italic; } /* 기울임 */ .oblique { font-style: oblique; }

normal - 일반 텍스트

italic - 이탤릭 텍스트

oblique - 기울임 텍스트

5. line-height (줄 간격)

줄과 줄 사이의 간격을 설정합니다.

/* 숫자 (배수) */ .text { line-height: 1.5; /* 글자 크기의 1.5배 */ } /* 픽셀 */ .text { line-height: 24px; } /* 퍼센트 */ .text { line-height: 150%; } /* normal (기본값) */ .text { line-height: normal; }

line-height: 1.2 - 줄 간격이 좁습니다. 여러 줄의 텍스트가 있을 때 줄 간격이 좁게 표시됩니다.

line-height: 1.8 - 줄 간격이 넓습니다. 여러 줄의 텍스트가 있을 때 줄 간격이 넓게 표시되어 읽기 편합니다.

6. text-decoration (텍스트 장식)

텍스트에 밑줄, 취소선 등을 추가합니다.

/* 밑줄 */ .underline { text-decoration: underline; } /* 취소선 */ .line-through { text-decoration: line-through; } /* 윗줄 */ .overline { text-decoration: overline; } /* 없음 */ .none { text-decoration: none; } /* 색상과 스타일 지정 */ .custom { text-decoration: underline wavy red; }

underline - 밑줄

line-through - 취소선

overline - 윗줄

wavy underline - 물결 밑줄

none - 링크 밑줄 제거

7. text-transform (대소문자 변환)

텍스트의 대소문자를 변환합니다.

/* 대문자 */ .uppercase { text-transform: uppercase; } /* 소문자 */ .lowercase { text-transform: lowercase; } /* 첫 글자만 대문자 */ .capitalize { text-transform: capitalize; } /* 없음 */ .none { text-transform: none; }

uppercase - 모든 글자를 대문자로

LOWERCASE - 모든 글자를 소문자로

capitalize - 각 단어의 첫 글자만 대문자로

8. letter-spacing과 word-spacing

글자 간격과 단어 간격을 조절합니다.

/* 글자 간격 */ .letter-spacing { letter-spacing: 2px; } /* 단어 간격 */ .word-spacing { word-spacing: 10px; }

letter-spacing: 2px - 글자 간격이 넓습니다

letter-spacing: -1px - 글자 간격이 좁습니다

word spacing: 10px - 단어 간격이 넓습니다
