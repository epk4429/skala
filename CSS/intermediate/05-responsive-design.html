05. 반응형 디자인 (Responsive Design)

1. 미디어 쿼리 (Media Queries)

화면 크기에 따라 다른 스타일을 적용합니다.

/* 기본 스타일 */ .container { width: 100%; } /* 태블릿 (768px 이상) */ @media (min-width: 768px) { .container { width: 750px; } } /* 데스크톱 (1024px 이상) */ @media (min-width: 1024px) { .container { width: 1000px; } } /* 모바일 (768px 미만) */ @media (max-width: 767px) { .menu { display: none; } }

2. viewport 메타 태그

모바일에서 올바르게 표시되도록 설정합니다.

<meta name="viewport" content="width=device-width, initial-scale=1.0"> /* 설명: - width=device-width: 화면 너비에 맞춤 - initial-scale=1.0: 초기 확대/축소 비율 */

3. 유동적 단위 사용

고정 크기 대신 유동적인 단위를 사용합니다.

/* 퍼센트 */ .width-percent { width: 100%; } /* vw, vh (뷰포트 기준) */ .width-viewport { width: 50vw; /* 뷰포트 너비의 50% */ height: 50vh; /* 뷰포트 높이의 50% */ } /* rem (루트 요소 기준) */ .font-size { font-size: 1.5rem; } /* em (부모 요소 기준) */ .font-size-em { font-size: 1.2em; }

4. Flexbox와 Grid의 반응형 활용

Flexbox와 Grid를 활용한 반응형 레이아웃입니다.

/* Flexbox 반응형 */ .container { display: flex; flex-wrap: wrap; } .item { flex: 1 1 200px; /* 최소 200px, 가능한 만큼 늘어남 */ } /* Grid 반응형 */ .container { display: grid; grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); }
