01. CSS 선택자 (Selectors)

1. 기본 선택자

요소를 선택하는 가장 기본적인 방법들입니다.

/* 태그 선택자 - 모든 p 태그에 적용 */ p { color: blue; } /* 클래스 선택자 - .클래스명 */ .my-class { color: red; } /* ID 선택자 - #아이디명 */ #my-id { color: green; }

이것은 p 태그입니다 (태그 선택자)

이것은 클래스를 가진 p 태그입니다

이것은 ID를 가진 p 태그입니다

2. 자손 선택자와 자식 선택자

요소의 계층 구조를 이용하여 선택합니다.

/* 자손 선택자 (Descendant) - 모든 하위 요소 */ div p { color: blue; } /* 자식 선택자 (Child) - 직접 자식만 */ div > p { color: red; }

div의 직접 자식 p (빨간색)

div의 손자 p (파란색)

3. 형제 선택자

같은 부모를 가진 형제 요소를 선택합니다.

/* 인접 형제 선택자 - 바로 다음 형제 */ h2 + p { color: blue; } /* 일반 형제 선택자 - 모든 다음 형제 */ h2 ~ p { background: yellow; }

제목

h2 바로 다음 p (파란색, 노란 배경)

h2 다음 p (노란 배경만)

4. 속성 선택자

요소의 속성을 기준으로 선택합니다.

/* 특정 속성을 가진 요소 */ a[href] { color: blue; } /* 속성 값이 정확히 일치 */ input[type="text"] { border: 2px solid blue; } /* 속성 값에 특정 문자열 포함 */ a[href*="example"] { font-weight: bold; }
예제 링크

5. 가상 클래스 선택자

요소의 상태나 위치를 기준으로 선택합니다.

/* 호버 상태 */ button:hover { background: blue; } /* 첫 번째 자식 */ li:first-child { color: red; } /* 마지막 자식 */ li:last-child { color: blue; } /* n번째 자식 */ li:nth-child(2) { color: green; }
  • 첫 번째 항목 (빨간색)
  • 두 번째 항목 (초록색)
  • 세 번째 항목
  • 마지막 항목 (파란색)

6. 가상 요소 선택자

요소의 특정 부분을 선택합니다.

/* 요소 앞에 내용 추가 */ p::before { content: "앞: "; color: red; } /* 요소 뒤에 내용 추가 */ p::after { content: " (뒤)"; color: blue; } /* 첫 줄 선택 */ p::first-line { font-weight: bold; }

이 문단은 가상 요소를 사용합니다.

7. 선택자 조합

여러 선택자를 조합하여 더 정확하게 선택합니다.

/* 여러 클래스 동시 적용 */ .btn.primary { background: blue; } /* 그룹 선택자 */ h1, h2, h3 { color: darkblue; } /* 복합 선택자 */ div.container > p.intro { font-size: 18px; }
