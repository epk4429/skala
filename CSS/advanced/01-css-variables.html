01. CSS 변수 (Custom Properties)

1. CSS 변수 정의

재사용 가능한 값을 변수로 정의합니다.

/* :root에 전역 변수 정의 */ :root { --primary-color: #2196F3; --spacing: 20px; --font-size: 16px; } /* 변수 사용 */ .element { color: var(--primary-color); padding: var(--spacing); font-size: var(--font-size); } /* 기본값 설정 */ .element { color: var(--primary-color, #000); /* 변수가 없으면 #000 사용 */ }
CSS 변수를 사용한 스타일

2. 변수 상속과 범위

변수는 상속되며 특정 요소에만 정의할 수 있습니다.

/* 전역 변수 */ :root { --color: blue; } /* 특정 요소에만 적용 */ .container { --color: red; } /* 자식 요소에서 사용 */ .child { color: var(--color); /* container의 --color 사용 */ }

3. JavaScript로 변수 제어

JavaScript로 CSS 변수를 동적으로 변경할 수 있습니다.

// 변수 값 가져오기 const root = document.documentElement; const color = getComputedStyle(root).getPropertyValue('--primary-color'); // 변수 값 설정하기 root.style.setProperty('--primary-color', '#FF5722'); // 변수 값 변경하기 document.querySelector('.element').style.setProperty('--spacing', '30px');
