01. 변수와 데이터 타입

1. 변수 선언하기

JavaScript에서 변수를 선언하는 방법은 3가지가 있습니다:

  • var: 오래된 방식 (현재는 거의 사용하지 않음)
  • let: 값이 변경될 수 있는 변수
  • const: 값이 변경되지 않는 상수

2. 데이터 타입 확인하기

JavaScript의 기본 데이터 타입을 확인해봅시다.

3. 타입 변환하기

문자열과 숫자 사이의 변환 방법을 알아봅시다.

4. 템플릿 리터럴 (Template Literal)

백틱(`)을 사용하여 문자열과 변수를 쉽게 결합할 수 있습니다.
