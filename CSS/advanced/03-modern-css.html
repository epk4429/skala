03. 모던 CSS 기법

1. backdrop-filter (배경 필터)

요소 뒤의 배경에 필터 효과를 적용합니다.

/* 블러 효과 */ .glass { background: rgba(255, 255, 255, 0.2); backdrop-filter: blur(10px); } /* 밝기 조절 */ .bright { backdrop-filter: brightness(150%); }
Glass morphism 효과

2. clip-path (클리핑)

요소의 표시 영역을 자릅니다.

/* 삼각형 */ .triangle { clip-path: polygon(50% 0%, 0% 100%, 100% 100%); } /* 원 */ .circle { clip-path: circle(50%); } /* 커스텀 모양 */ .custom { clip-path: polygon(0% 0%, 100% 0%, 100% 75%, 75% 100%, 0% 100%); }

3. CSS Grid 고급

Grid의 고급 기능을 활용합니다.

/* 자동 배치 */ .container { display: grid; grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); gap: 20px; } /* 그리드 영역 이름 */ .container { grid-template-areas: "header header header" "sidebar main main" "footer footer footer"; } .header { grid-area: header; } .sidebar { grid-area: sidebar; } .main { grid-area: main; } .footer { grid-area: footer; }

4. 컨테이너 쿼리 (Container Queries)

부모 요소 크기에 따라 스타일을 변경합니다.

/* 컨테이너 정의 */ .container { container-type: inline-size; } /* 컨테이너 쿼리 */ @container (min-width: 400px) { .card { display: grid; grid-template-columns: 1fr 1fr; } }
