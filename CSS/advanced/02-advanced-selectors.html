02. 고급 선택자

1. :not() 선택자

특정 요소를 제외하고 선택합니다.

/* p 태그 중 class가 없는 것만 */ p:not(.special) { color: blue; } /* 버튼 중 disabled가 아닌 것만 */ button:not(:disabled) { cursor: pointer; }

2. :nth-child() 고급 사용

복잡한 패턴으로 요소를 선택합니다.

/* 짝수 번째 */ li:nth-child(even) { background: #f0f0f0; } /* 홀수 번째 */ li:nth-child(odd) { background: #fff; } /* 3의 배수 */ li:nth-child(3n) { color: red; } /* 3n+1 (1, 4, 7, ...) */ li:nth-child(3n+1) { font-weight: bold; } /* 마지막에서 n번째 */ li:nth-last-child(2) { border-bottom: 2px solid blue; }

3. 속성 선택자 고급

속성 값의 패턴으로 선택합니다.

/* 시작하는 값 */ a[href^="https"] { color: green; } /* 끝나는 값 */ a[href$=".pdf"] { color: red; } /* 포함하는 값 */ a[href*="example"] { font-weight: bold; } /* 공백으로 구분된 단어 */ div[class~="important"] { border: 2px solid red; }

4. :is()와 :where() 선택자

여러 선택자를 그룹화합니다.

/* :is() - 우선순위 유지 */ :is(h1, h2, h3) { color: blue; } /* :where() - 우선순위 0 */ :where(h1, h2, h3) { margin: 0; }
