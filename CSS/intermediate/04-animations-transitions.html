04. 애니메이션과 트랜지션

1. transition (트랜지션)

속성 변화를 부드럽게 만듭니다.

/* 기본 사용법 */ .element { transition: property duration timing-function delay; } /* 예제 */ .box { width: 100px; transition: width 0.3s ease; } .box:hover { width: 200px; } /* 여러 속성 */ .box { transition: width 0.3s, background 0.5s; } /* 모든 속성 */ .box { transition: all 0.3s ease; }

마우스를 올려보세요 (hover)

2. animation (애니메이션)

키프레임을 사용한 복잡한 애니메이션입니다.

/* 키프레임 정의 */ @keyframes slide { from { transform: translateX(0); } to { transform: translateX(200px); } } /* 애니메이션 적용 */ .element { animation: slide 2s infinite alternate; } /* 애니메이션 속성 */ .element { animation-name: slide; animation-duration: 2s; animation-iteration-count: infinite; animation-direction: alternate; animation-timing-function: ease; }

자동으로 애니메이션이 실행됩니다

3. transform (변형)

요소를 변형시킵니다.

/* 이동 */ transform: translateX(50px); transform: translateY(50px); transform: translate(50px, 50px); /* 회전 */ transform: rotate(45deg); /* 크기 조절 */ transform: scale(1.5); transform: scaleX(2); transform: scaleY(0.5); /* 기울임 */ transform: skewX(20deg); transform: skewY(10deg); /* 조합 */ transform: translate(50px, 50px) rotate(45deg) scale(1.2);
