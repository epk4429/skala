01. Flexbox (Flexible Box Layout)

1. Flexbox 기본 개념

Flexbox는 요소들을 유연하게 배치하는 레이아웃 방식입니다.

/* Flexbox 컨테이너 만들기 */ .container { display: flex; } /* 주요 개념: - flex container: display: flex를 가진 부모 요소 - flex items: 컨테이너의 직접 자식 요소들 - main axis: 주축 (기본값: 가로) - cross axis: 교차축 (기본값: 세로) */

2. flex-direction (방향)

주축의 방향을 설정합니다.

/* 가로 방향 (기본값) */ .container { flex-direction: row; } /* 세로 방향 */ .container { flex-direction: column; } /* 가로 역방향 */ .container { flex-direction: row-reverse; } /* 세로 역방향 */ .container { flex-direction: column-reverse; }

3. justify-content (주축 정렬)

주축(main axis) 방향으로 요소들을 정렬합니다.

/* 시작점 정렬 */ .container { justify-content: flex-start; } /* 끝점 정렬 */ .container { justify-content: flex-end; } /* 중앙 정렬 */ .container { justify-content: center; } /* 양쪽 정렬 (간격 동일) */ .container { justify-content: space-between; } /* 주변 간격 동일 */ .container { justify-content: space-around; } /* 모든 간격 동일 */ .container { justify-content: space-evenly; }

4. align-items (교차축 정렬)

교차축(cross axis) 방향으로 요소들을 정렬합니다.

/* 시작점 정렬 */ .container { align-items: flex-start; } /* 끝점 정렬 */ .container { align-items: flex-end; } /* 중앙 정렬 */ .container { align-items: center; } /* 기준선 정렬 */ .container { align-items: baseline; } /* 늘리기 */ .container { align-items: stretch; }

5. flex-wrap (줄바꿈)

요소들이 한 줄에 들어가지 않을 때 줄바꿈을 설정합니다.

/* 줄바꿈 안 함 (기본값) */ .container { flex-wrap: nowrap; } /* 줄바꿈 허용 */ .container { flex-wrap: wrap; } /* 역순 줄바꿈 */ .container { flex-wrap: wrap-reverse; }

6. flex 속성 (자식 요소)

flex item의 크기와 성장/축소 비율을 설정합니다.

/* flex 축약형 */ .item { flex: 1; /* flex-grow: 1, flex-shrink: 1, flex-basis: 0% */ } /* 개별 속성 */ .item { flex-grow: 1; /* 남은 공간 차지 비율 */ flex-shrink: 1; /* 공간 부족 시 축소 비율 */ flex-basis: 200px; /* 기본 크기 */ } /* 고정 크기 */ .item { flex: 0 0 200px; /* grow: 0, shrink: 0, basis: 200px */ }
