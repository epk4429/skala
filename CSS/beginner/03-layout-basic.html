03. 레이아웃 기본

1. display 속성

요소가 화면에 표시되는 방식을 결정합니다.

/* block - 블록 요소 (한 줄 전체 차지) */ div { display: block; /* width, height, margin, padding 모두 사용 가능 */ } /* inline - 인라인 요소 (콘텐츠만큼만 차지) */ span { display: inline; /* width, height 사용 불가 */ /* margin, padding은 좌우만 가능 */ } /* inline-block - 인라인처럼 배치, 블록처럼 크기 조절 */ .button { display: inline-block; /* width, height 사용 가능 */ /* 인라인처럼 나란히 배치 */ } /* none - 요소 숨김 */ .hidden { display: none; }

2. 텍스트 정렬

텍스트와 인라인 요소의 정렬을 설정합니다.

/* 왼쪽 정렬 (기본값) */ .text-left { text-align: left; } /* 가운데 정렬 */ .text-center { text-align: center; } /* 오른쪽 정렬 */ .text-right { text-align: right; } /* 양쪽 정렬 */ .text-justify { text-align: justify; }

왼쪽 정렬 (left)

가운데 정렬 (center)

오른쪽 정렬 (right)

양쪽 정렬 (justify) - 긴 텍스트에서 줄 끝을 맞춥니다.

3. float 속성

요소를 왼쪽 또는 오른쪽으로 띄웁니다.

/* 왼쪽으로 띄움 */ .float-left { float: left; } /* 오른쪽으로 띄움 */ .float-right { float: right; } /* float 해제 */ .clearfix::after { content: ""; display: table; clear: both; }
float: left

이 텍스트는 float된 요소 주변으로 흐릅니다. float는 레이아웃을 만들 때 사용되지만, 현대적인 방법으로는 Flexbox나 Grid를 사용하는 것이 권장됩니다.

float: right

오른쪽으로 float된 요소입니다.

4. 중앙 정렬 방법

요소를 가로/세로로 중앙에 배치하는 방법들입니다.

/* 방법 1: margin auto (가로 중앙) */ .center-horizontal { width: 300px; margin: 0 auto; } /* 방법 2: text-align (인라인 요소) */ .parent { text-align: center; } /* 방법 3: line-height (한 줄 텍스트 세로 중앙) */ .single-line { height: 100px; line-height: 100px; } /* 방법 4: padding (간단한 세로 중앙) */ .vertical-center { padding: 50px 0; }
margin: 0 auto (가로 중앙)
text-align: center
line-height로 세로 중앙

5. overflow 속성

콘텐츠가 요소 크기를 넘칠 때 처리 방법을 설정합니다.

/* visible - 넘치는 부분 표시 (기본값) */ .visible { overflow: visible; } /* hidden - 넘치는 부분 숨김 */ .hidden { overflow: hidden; } /* scroll - 스크롤바 항상 표시 */ .scroll { overflow: scroll; } /* auto - 필요할 때만 스크롤바 표시 */ .auto { overflow: auto; }
overflow: hidden - 넘치는 부분이 잘립니다. 이 텍스트는 보이지 않을 것입니다.
overflow: scroll - 스크롤바가 항상 표시됩니다. 이 텍스트는 스크롤하여 볼 수 있습니다.
overflow: auto - 필요할 때만 스크롤바가 나타납니다.

6. visibility 속성

요소의 표시 여부를 제어합니다.

/* visible - 표시 (기본값) */ .visible { visibility: visible; } /* hidden - 숨김 (공간은 차지함) */ .hidden { visibility: hidden; } /* display: none과의 차이 */ /* display: none - 공간도 차지하지 않음 */ /* visibility: hidden - 공간은 차지하지만 보이지 않음 */
