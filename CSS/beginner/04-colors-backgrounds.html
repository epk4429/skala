04. 색상과 배경

1. 색상 지정 방법

CSS에서 색상을 지정하는 여러 가지 방법이 있습니다.

/* 1. 색상 이름 */ color: red; color: blue; color: green; /* 2. HEX 코드 (16진수) */ color: #FF0000; /* 빨간색 */ color: #00FF00; /* 초록색 */ color: #0000FF; /* 파란색 */ color: #FF5733; /* 주황색 */ color: #333; /* #333333 축약형 */ /* 3. RGB */ color: rgb(255, 0, 0); /* 빨간색 */ color: rgb(0, 255, 0); /* 초록색 */ color: rgb(0, 0, 255); /* 파란색 */ /* 4. RGBA (투명도 포함) */ color: rgba(255, 0, 0, 0.5); /* 반투명 빨간색 */ background: rgba(0, 0, 0, 0.3); /* 반투명 검은 배경 */ /* 5. HSL */ color: hsl(0, 100%, 50%); /* 빨간색 */ color: hsl(120, 100%, 50%); /* 초록색 */ color: hsl(240, 100%, 50%); /* 파란색 */ /* 6. HSLA (투명도 포함) */ color: hsla(0, 100%, 50%, 0.5);
색상 이름
HEX
RGB
RGBA
HSL

2. background-color (배경색)

요소의 배경색을 설정합니다.

/* 단색 배경 */ .box { background-color: #3498db; } /* 축약형 */ .box { background: #3498db; }
배경색: #3498db
반투명 배경색: rgba(46, 204, 113, 0.5)

3. background-image (배경 이미지)

요소의 배경에 이미지를 설정합니다.

/* 배경 이미지 설정 */ .box { background-image: url('image.jpg'); } /* 배경 이미지 + 색상 (이미지 로딩 실패 시 대체) */ .box { background-color: #f0f0f0; background-image: url('image.jpg'); } /* 여러 배경 이미지 */ .box { background-image: url('image1.jpg'), url('image2.jpg'); }
그라데이션 배경 (이미지 대신)

4. background-repeat (반복)

배경 이미지의 반복 방식을 설정합니다.

/* 반복 (기본값) */ .box { background-repeat: repeat; } /* 반복 안 함 */ .box { background-repeat: no-repeat; } /* 가로 반복 */ .box { background-repeat: repeat-x; } /* 세로 반복 */ .box { background-repeat: repeat-y; }
반복 패턴 (시각적 예시)

5. background-position (위치)

배경 이미지의 위치를 설정합니다.

/* 키워드 사용 */ .box { background-position: center; background-position: top left; background-position: bottom right; } /* 픽셀 값 사용 */ .box { background-position: 50px 100px; /* 가로 세로 */ } /* 퍼센트 사용 */ .box { background-position: 50% 50%; /* 중앙 정렬 */ }
background-position: center

6. background-size (크기)

배경 이미지의 크기를 설정합니다.

/* 원본 크기 (기본값) */ .box { background-size: auto; } /* 고정 크기 */ .box { background-size: 200px 100px; } /* 퍼센트 */ .box { background-size: 50% 50%; } /* cover - 영역을 완전히 덮음 */ .box { background-size: cover; } /* contain - 이미지가 잘리지 않도록 */ .box { background-size: contain; }
background-size: cover

7. 그라데이션 (Gradient)

색상이 부드럽게 변하는 배경을 만듭니다.

/* 선형 그라데이션 */ .box { background: linear-gradient(to right, red, blue); background: linear-gradient(45deg, red, blue); background: linear-gradient(to bottom, red, yellow, blue); } /* 방사형 그라데이션 */ .box { background: radial-gradient(circle, red, blue); background: radial-gradient(ellipse at center, red, blue); } /* 반복 그라데이션 */ .box { background: repeating-linear-gradient(45deg, red, red 10px, blue 10px, blue 20px); }
선형 그라데이션 (좌→우)
선형 그라데이션 (45도)
방사형 그라데이션
