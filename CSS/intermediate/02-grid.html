02. CSS Grid

1. Grid 기본 개념

Grid는 2차원 레이아웃 시스템입니다.

/* Grid 컨테이너 만들기 */ .container { display: grid; } /* 주요 개념: - grid container: display: grid를 가진 부모 요소 - grid items: 컨테이너의 직접 자식 요소들 - grid lines: 그리드 선 (행과 열을 구분) - grid tracks: 그리드 트랙 (행 또는 열) - grid cells: 그리드 셀 (한 칸) */

2. grid-template-columns와 grid-template-rows

그리드의 열과 행 크기를 정의합니다.

/* 고정 크기 */ .container { grid-template-columns: 200px 200px 200px; grid-template-rows: 100px 100px; } /* fr 단위 (비율) */ .container { grid-template-columns: 1fr 2fr 1fr; /* 1:2:1 비율 */ } /* repeat() 함수 */ .container { grid-template-columns: repeat(3, 1fr); /* 3개의 동일한 열 */ } /* minmax() 함수 */ .container { grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); }

3. gap (간격)

그리드 항목 사이의 간격을 설정합니다.

/* 행과 열 간격 동일 */ .container { gap: 20px; } /* 행과 열 간격 개별 설정 */ .container { row-gap: 20px; column-gap: 30px; }

4. grid-column과 grid-row

그리드 항목의 위치를 지정합니다.

/* 열 범위 지정 */ .item { grid-column: 1 / 3; /* 1번부터 3번 전까지 (2칸) */ grid-column: span 2; /* 2칸 차지 */ } /* 행 범위 지정 */ .item { grid-row: 1 / 3; grid-row: span 2; }
