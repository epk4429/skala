03. Positioning (위치 지정)

1. position 속성 종류

요소의 위치를 지정하는 방법입니다.

/* static - 기본값 (문서 흐름에 따름) */ .element { position: static; } /* relative - 상대 위치 (원래 위치 기준) */ .element { position: relative; top: 20px; left: 30px; } /* absolute - 절대 위치 (가장 가까운 positioned 부모 기준) */ .element { position: absolute; top: 0; right: 0; } /* fixed - 고정 위치 (뷰포트 기준) */ .element { position: fixed; top: 0; right: 0; } /* sticky - 스크롤 시 고정 */ .element { position: sticky; top: 0; }

2. relative (상대 위치)

원래 위치를 기준으로 이동합니다.

relative - top: 20px, left: 30px

3. absolute (절대 위치)

가장 가까운 positioned 부모를 기준으로 위치합니다.

absolute - top: 10px, right: 10px

4. fixed (고정 위치)

뷰포트를 기준으로 고정됩니다.

/* 화면에 고정되는 요소 */ .fixed-element { position: fixed; bottom: 20px; right: 20px; /* 스크롤해도 같은 위치에 고정 */ }

5. z-index (층 순서)

요소의 겹치는 순서를 제어합니다.

z-index: 1
z-index: 2 (위에)
