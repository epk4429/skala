02. 박스 모델 (Box Model)

1. 박스 모델의 구성 요소

모든 HTML 요소는 박스 모델로 구성되어 있습니다.

/* 박스 모델의 4가지 영역 */ 요소 { width: 200px; /* 콘텐츠 너비 */ height: 100px; /* 콘텐츠 높이 */ padding: 20px; /* 안쪽 여백 */ border: 5px solid; /* 테두리 */ margin: 20px; /* 바깥쪽 여백 */ } /* 구조: margin (바깥 여백) └─ border (테두리) └─ padding (안쪽 여백) └─ content (콘텐츠) */
콘텐츠

2. width와 height

요소의 너비와 높이를 설정합니다.

/* 고정 크기 */ .box1 { width: 300px; height: 200px; } /* 퍼센트 크기 (부모 요소 기준) */ .box2 { width: 50%; height: 50%; } /* 최대/최소 크기 */ .box3 { min-width: 200px; max-width: 500px; }
고정 크기: 300px × 100px
퍼센트 크기: 50% × 80px

3. padding (안쪽 여백)

콘텐츠와 테두리 사이의 여백입니다.

/* 모든 방향 동일 */ .box { padding: 20px; } /* 개별 설정 */ .box { padding-top: 10px; padding-right: 20px; padding-bottom: 30px; padding-left: 40px; } /* 축약형 (시계방향) */ .box { padding: 10px 20px 30px 40px; /* 위 오른쪽 아래 왼쪽 */ } /* 축약형 (상하 좌우) */ .box { padding: 10px 20px; /* 상하 좌우 */ }
padding: 20px (모든 방향)
padding: 10px 30px (상하 좌우)

4. border (테두리)

요소의 테두리를 설정합니다.

/* 기본 테두리 */ .box { border: 2px solid black; } /* 개별 속성 */ .box { border-width: 3px; border-style: solid; border-color: blue; } /* 방향별 설정 */ .box { border-top: 2px solid red; border-right: 3px dashed blue; border-bottom: 4px dotted green; border-left: 5px double orange; } /* 테두리 스타일 종류 */ .solid { border-style: solid; } /* 실선 */ .dashed { border-style: dashed; } /* 점선 */ .dotted { border-style: dotted; } /* 점 */ .double { border-style: double; } /* 이중선 */ .none { border-style: none; } /* 없음 */
solid (실선)
dashed (점선)
dotted (점)
double (이중선)

5. margin (바깥쪽 여백)

요소와 다른 요소 사이의 여백입니다.

/* 모든 방향 동일 */ .box { margin: 20px; } /* 개별 설정 */ .box { margin-top: 10px; margin-right: 20px; margin-bottom: 30px; margin-left: 40px; } /* 축약형 */ .box { margin: 10px 20px 30px 40px; /* 위 오른쪽 아래 왼쪽 */ } /* 중앙 정렬 */ .box { margin: 0 auto; /* 좌우 중앙 */ } /* 음수 마진 */ .box { margin: -10px; /* 요소를 위로 당김 */ }
margin: 20px
margin: 10px 50px (상하 좌우)
margin: 0 auto (중앙 정렬)

6. box-sizing 속성

박스 크기 계산 방식을 변경합니다.

/* content-box (기본값) */ /* width/height = 콘텐츠 크기만 */ .box1 { width: 200px; padding: 20px; border: 5px solid; /* 실제 너비 = 200 + 40 + 10 = 250px */ } /* border-box */ /* width/height = 콘텐츠 + padding + border */ .box2 { box-sizing: border-box; width: 200px; padding: 20px; border: 5px solid; /* 실제 너비 = 200px (전체) */ }
content-box (기본값)
실제 너비: 250px
border-box
실제 너비: 200px
